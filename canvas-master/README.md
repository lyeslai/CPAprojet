# CPA

## Pour lancer le projet

### Prérequis

- nodejs

### Installation

```bash
npm install
```

ou avec yarn

```bash
yarn install
```

ou à la PPTI

```bash
./yarn.sh
```

### Lancer le projet

```bash
npm run start
```

ou avec yarn

```bash
yarn start
```

ou à la PPTI

```bash
./start.sh
```
